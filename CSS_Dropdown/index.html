A Simple Ugly CSS Dropdown Menu

This was made using mostly W3Schools, Stack Overflow, and whatever else I could possibly google.

I did sneak a peak at Moshe Plotkin's website to find the "position: absolute" CSS attribute but that's all.

It was designed using unordered lists, with list items as the links, given that it wasn't meant to be functional. A small bit of repurposing however can make it functional.

A better knowledge of CSS can make it beautiful. Feel free to use it as a template.
